How to open the app:
*Follow first step & final step below.*


a) Set-up Using Virtual Environment

1. navigate to file path
	# cd Users\Admin\Desktop\ocr_app
2. create a virtual environment in cmd
	# py -m venv ocr_app_env#
3. activate virtual environment
	# ocr_app_env\Scripts\activate
4. Install packages
	# py -m pip install PyQt6 easyocr pillow piexif
5. Run the app
	# py main.py


b) First Time Set-up

Step 1- Go to file directory under ocr_app in cmd 	<-- cd desktop\ocr_app -->
Step 2- Download necessary files and packages		<-- py -m pip install PyQt6 easyocr pillow piexif -->
Step 3- Wait for download to finish
Step 4- Run the app					<-- py main.py -->
